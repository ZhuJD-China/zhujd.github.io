Python面向对象（一切皆对象）

2020-6-6 13:03

当你需要编写一个大型程序或面对某一更适合此方法的问题时，你可以考虑使用面向对象式的编程技术。 类与对象是面向对象编程的两个主要方面。一个类（Class）能够创建一种新的类型（Type）， 其中对象（Object）就是类的实例（Instance）。可以这样来类比：你可以拥有类型 int 的变量，也就是说存储整数的变量是 int 类的实例（对象）
